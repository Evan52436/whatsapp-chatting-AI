# whatsapp-chatting-AI
## THIS BOT IS OPTIMIZED TO RUN ON LINUX MACHINE
please use wisely. to start : <br>
1. Clone the Repository
2. Generate .env file
3. Edit the following .env file like this :
```GROQ_API_KEY=YOURAPIKEY```
4. Modify the ```index.js``` edit line 16 <br>
```const SYSTEM_PROMPT```
5. Save the file
6. run ```npm install``` on your terminal
7. run ```node index.js``` and enjoy
